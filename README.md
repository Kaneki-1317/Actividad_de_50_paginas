# 📚 Taller de Maquetación Web

## 📌 Descripción

Este proyecto consiste en el desarrollo de **páginas web** como práctica de maquetación, utilizando tecnologías fundamentales del desarrollo frontend.

El objetivo es mejorar habilidades en **HTML, CSS y diseño responsivo**, aplicando buenas prácticas de estructura, estilo y organización.

---

## 🎯 Objetivos

* Dominar la estructura semántica en HTML5
* Aplicar estilos con CSS3
* Implementar diseños modernos y responsivos
* Organizar múltiples páginas dentro de un mismo proyecto
* Mejorar la presentación visual de interfaces web

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Responsive Design (Media Queries)

Cada página incluye:

* Header (encabezado)
* Navegación
* Sección principal (contenido)
* Footer (pie de página)

Las páginas pueden variar en diseño como:

* Landing pages
* Portafolios
* Blogs
* Páginas de productos
* Formularios

---

## 🚀 Cómo ejecutar el proyecto

1. Clona este repositorio:

```
git clone https://github.com/tu-usuario/taller-maquetacion.git
```

2. Abre la carpeta del proyecto

3. Ejecuta el archivo:

```
index.html
```
